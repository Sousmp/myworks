Антизубрёжка

Для запоминания английских слов мы используем метод интервальных повторений, при котором повторение происходит через неравные промежутки времени.

Этот метод можно назвать методом «Антизубрежка». Никакие другие методики запоминания не дают такого результата!

Так, при изучении блока из 20 слов сначала повторяется 7-8 раз, а затем – через 30 секунд. Затем повторения проводятся через все более увеличивающиеся промежутки времени. Таким образом, достигается 97% запоминание за 3 дня при минимальном числе повторений.
